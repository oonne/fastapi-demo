# fastapi-demo
FastApi DEMO

## 使用

1. 创建虚拟环境：
```bash
python3 -m venv venv
```

2. 激活虚拟环境：
```bash
# macOS/Linux:
source venv/bin/activate
# Windows:
venv\Scripts\activate
```

3. 安装依赖：
```bash
pip install -r requirements.txt
```

## 开发

### 导出依赖列表（安装新包后记得更新）：
```bash
pip freeze > requirements.txt
```
